# Descrption
A simple implementation of cellular automata such as (Conway's Game of Life and elementary cellular automata)
- I tested and compiled on linux

## Dependencies
- gcc (GNU C compiler)
- sh ( shell scripting language)
- SDL2 (A graphics library for C)

## Compile and run
```console
./build.sh
./main

```
